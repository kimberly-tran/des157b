Arrow Functions Example: Change Color

This simple function I created stores a color inputted as a string and assigns it to a variable called "color". This is then used when changing the color value of the h1 tag.

Named Function

            function changeColor(aString){
                color = aString;
                document.querySelector('h1').style.color = color;
            }

Function Expression

            const changeColor = function(aString){
                color = aString;
                document.querySelector('h1').style.color = color;
            }

Arrow Function

            const changeColor = aString => {
                color = aString;
                document.querySelector('h1').style.color = color;
            }
